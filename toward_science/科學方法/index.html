    科學方法¶

    本文章以非主流的數學公理系統定義了科學方法，同時探討了一些科學方法的運用情況。假設一個變數在時間、空間、其他條件的改變之下也隨著改變，科學探討的是其中的過程和原理。例如時間、空間位置的改變怎樣影響到該變數。當有很多環境條件的時候，我們需要借助線性代數來分析，否則只是胡鬧。

    緣起¶

    我發現Google很難找到「科學方法(scientific method)」的解說。反映了中文使用者普遍的科學教育程度極低，就算有人想學、想識，也很難找到合適可信的資料，因此我認為有必要寫一篇文來解釋科學方法。

    變數¶

    假設一個A現象是一個變數，會因為某些環境因素而呈現新的形象。

    A現象所發生在的空間有很多環境因素，例如其自身的位置、其他現象的位置、現象發生的時間，我們猜測這些環境因素的改變會令A現象變樣。

    - 為甚麼要這樣假設？
    - 因為要先定義一個科學的研究對象

- 為甚麼一定要是變數？
    - 先假設是一個變數。如果後來發現了無論環境因素怎樣改變、A現象也是恆定的話，這也是一個新發現。

- 甚麼是環境因素？
    - 該現象本體以外的所有東西。邏輯地說就是subset[ not(該現象) ]。

    相互作用¶

    我們關心的有三件事：

    Text Only
    1
2
3
    1. 如果A現象在物理空間內的位置改變了，它會不會也因此而改變？
2. 如果A現象發生的時間改變了，它會不會也因此而改變？
3. 如果周圍的其他現象改變了，A現象會不會也因此而改變？

    我們關心的是「相互作用」。

    Text Only
    1
2
3
    A現象 <==> 物理空間
A現象 <==> 時間
A現象 <==> 其他現象

    當我們將物理空間和時間也看待成為一種現象的話，我們也可以這樣問：

    Text Only
    1
2
    1. 如果A現象改變了，物理空間會不會也因此而改變？
2. 如果A現象改變了，時間會不會也因此而改變？

    因此，相互作用就變成簡單一個：

    Text Only
    1
    A現象 <==> 其他現象

    集合¶

    如果A現象和B現象的性質一樣的話，我們可以把它們組合成一個類型。

    Text Only
    1
2
    A類型 =: {A現象, B現象}
C類型 =: {C現象, D現象}

    A類型的裏面包含了A現象和B現象。如是者我們看的相互作用會變成這樣：

    Text Only
    1
    A類型 <==> 其他類型

    A現象和C現象之間的相互作用，同時也發生在B現象和D現象之間。我們對A現象和C現象之間相互作用的理解，也可以適用於相同類型之間的相互作用。

    實驗與觀察¶

    我們想獲知現象之間的相互作用是如何發生的，以及其原理。為此，我們需要介入其中，刻意去令研究對象的現象之間發生相互作用，這行為就叫做實驗。

    例如，我們想知道A現象和時間之間的相互作用，我們可以維持其他所有環境因素都是恆定的狀態之下，**在不同的時間重覆**令A現象發生，然後觀察A現象有否改變。

    又例如，我們想知道A現象和物理空間之間的相互作用，我們可以維持其他所有環境因素都是恆定的狀態之下，**在不同的位置同時**令A現象發生，然後觀察A現象有否改變。

    如果無法靜止其他所有環境因素的話，實驗也可以進行，只是在分析結果的時候比較吃力而已。我們需要使用多變數的分析工具，尤其需要線性代數的知識(而不是人文學科的知識)。

    觀察也是一個現象¶

    觀察這個行為也是一個現象。如果觀察行為本身就能夠改變了A現象的性質，我們就要思考如何減輕觀察所帶來的效果。

    而實驗行為本身帶來的相互作用就是我們期望的東西，不過要確認的是實際施行的時候是不是跟原先計劃的一樣。

    沒有相互作用的現象¶

    如果Z現象與其他現象之間完全沒有任何相互作用的話，Z現象可被看待為不存在，事實上亦等同於不存在。原因如下：

    Text Only
    1
2
3
4
5
    1. 它不能夠影響任何東西。
    - 有沒有Z現象也沒有所謂。反正都是一樣。

2. 你無法得知Z現象存在。
    - Z現象無法干擾其他現象，因此我們沒有任何方法可以得知Z現象的存在。喪失了「得知」的這一個渠道。

    運用科學方法¶

    變數、現象、與其相互作用¶

    在地球表面上，物件會向下跌落。我們想知道是不是所有的地方都會同樣地發生「跌落」這個現象。因此我們做實驗，在不同的地方拿起物件然後放手，物件都一樣會跌落。同時拿起兩個物件，同時放手，一齊跌落。我們得知因此「跌落」的這個現象是不受物理空間和時間改變的影響。

    但是我們要仔細考慮所有實驗細節和環境因素和結果的細節。是不是每一次發生的「跌落」現象都是完全相同？例如我們必須要問，是不是每一次的跌落的**速度**、**加速率**都是一樣？

    同時我們又要問，「跌落」這個現象會不會影響到物理空間和時間本身？而物件本身的性質又會不會影響到「跌落」這個現象的過程和結果。

    科學能夠你說了算嗎？¶

    之所以各國團隊會重覆做同一個實驗，是因為要確定不是實驗結果不是因人而異、因時而異、因地而異。如果說一個幹細胞實驗，無論如何都無辦法重現當初的結果，排除實驗設計和過程的問題之後，餘下來就是造假的可能性了。

    如果他說那個原理是因地而異的話，然而他自己也不能夠在同一個實驗室重覆同一個實驗結果，問題就更大了。

    如果他說那個原理是因時而異的話，那麼我們也沒有辦法得知是甚麼時候才能夠令結果再現，很大機會根本那個只是一次性的現象，只發生在原始實驗當時的那一刻。如果如是的話，那麼這也就是「沒有相互作用的現象」。因為再也不可能發生第二次的，與「沒有這種東西」是一樣的。

    神鬼論證¶

    文科仔常見的「證明了世界上有鬼」的辯證法是這樣的。

    我有陰陽眼，經常見到鬼，你看不見但我看得見。科學儀器不能探測到鬼的存在，但我就是可以。

    如果一個現象有其他所有環境因素和實驗細節都是恆定的情況之下，只有在文科仔的情況有所改變，那麼我們要思考的是問題是否就出於文科仔的身上？

    文科仔是人，我也是人，大家也是人：

    Text Only
    1
    人 =: {我，你，他}

    假設人鬼之間有相互作用，如果大家都是人的情況之下，卻只有文科仔可以見到鬼，是不是文科仔出了問題？

    我們得知科學儀器的在一切物理環境的感知能力都比人類強，例如加速度、聲波、光波、電磁波、重力波，如果鬼的存在這一切的物理性質上都不能夠得以反映的話，就是說可能性只有一個：是文科仔出了問題。

    如果鬼與其他一切事情皆無相互作用，只有跟文科仔的認知之間有相互作用的話，答案之有兩個：

    1
2
    1. 這個文科仔是被選中的，值得解剖研究有甚麼特別的構造
2. 這個文科仔患上了思覺失調

    社會科學是科學嗎？¶

    不是。因為社會科學沒有努力去確保實驗環境的一致，即使不一致，也他們也沒有運用多變數分析手法和工具來分離各環境因素和結果之間的關係。最基本，線性代數是必修的。

    哲學在自然科學上的地位比物理、化學、生物學等高嗎？¶

    文科仔的老生常談是哲學超然於科學甚麼的，物理學及不上哲學甚麼的，又或者是哲學比自然科學更接近科學甚麼的。

    但即使(假設)哲學的研究方法符合了以上談論的科學方法，其研究對象也不是自然現象，哲學只是對其他哲學家的話語內容進行研究而已。

    物理、化學、生物學會根據自然現象設計實驗、驗證理論、獲得知識。哲學系在哪裡？對自然現象沒有相互作用的哲學，在自然科學的研究體系中不存在任何「地位」之類的東西。
